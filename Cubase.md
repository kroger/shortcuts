## General

| 0 | Mute/Unmute Objects  |
| 1 | Selection Tool       |
| 2 | Pencil (draw) Tool   |
| 3 | Erase Tool           |
| 4 | Split Tool           |
| 5 | Range Selection Tool |

| b | Toggle Lower Zone |
| l | Toggle Left Zone  |
| r | Toggle Right Zone |

| Return | Toggle Editors in Lower Zone (piano roll, sample) |
| c      | Open MixConsole in Lower Zone                     |
| d      | Open Drum Editor in Lower Zone                    |
